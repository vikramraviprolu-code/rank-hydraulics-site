# Rank Hydraulics Site

Standalone static website and quote landing page for Rank Hydraulics.

## Files

- `index.html` is the full website.
- `landing.html` is the focused quote landing page.
- `styles.css` contains the full design system and responsive layout.
- `script.js` handles the mobile menu, reveal animations and static email-form behavior.
- `assets/` contains local images and fonts used by the pages.
- `source-notes.md` records what was extracted from the original GitHub Pages site and what needs confirmation.

## Run Locally

Open `index.html` directly in a browser, or run:

```bash
npm run serve
```

Then visit `http://127.0.0.1:4173`.

No build step or dependency install is required.
